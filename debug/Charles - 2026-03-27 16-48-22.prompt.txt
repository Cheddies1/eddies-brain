## Default Behaviour in This Folder

When I reference a transcript:

1. Do not summarise lazily.
2. Prioritise:
   - Decisions made.
   - Action items (with owner if identifiable).
   - Risks raised.
   - Open questions.
   - Misalignment or tension.
3. Highlight contradictions or unclear commitments.
4. Separate fact from interpretation.

If something is uncertain:
- Do not stop the summary.
- State the uncertainty explicitly in the output.
- Prefer omission over invention.
- Use phrases like:
  - "Owner unclear from transcript"
  - "Attendance unclear from transcript"
  - "Mentioned, not confirmed present"

## Executive Summary

## Decisions

Only classify something as a Decision if the group clearly aligned or confirmed it.
If it is directional or likely but not explicitly agreed, place it under Strategic Observations instead.

## Action Items

Only include an action item if someone is clearly asked to do something, volunteers to do something, or the group explicitly agrees a next step.
Do not convert discussion points or likely next steps into actions unless the transcript supports it.
Do not infer ownership from context or seniority.

Format:
- Owner – Action – Context (if needed)

If owner unclear, state: Owner unclear.

## Risks / Concerns

Only real risks. No filler.

## Open Questions

## Strategic Observations (Optional)

Only include if strategically relevant:
- Misalignment

- Commercial vs technical tension
- Behavioural dynamics

Limit to 2–3 points maximum.
Avoid generic or abstract phrasing.

## Meeting Summary Discipline

For transcript-to-note generation:

- Summarise what was actually said, not what would be sensible to say.
- Preserve the distinction between:
  - agreed decisions
  - proposed approaches
  - open questions
  - background context
- If the group says "probably", "I think", "it depends", or "we need to clarify", do not write it up as settled.
- If only one person suggests something and nobody aligns, treat it as discussion, not decision.
- If a person is only mentioned historically or externally, do not list them as attending.
